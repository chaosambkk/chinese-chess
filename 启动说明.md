# 启动说明

## 快速开始

### 1. 安装依赖

在项目根目录运行：

```bash
npm run install-all
```

这会安装后端和前端的所有依赖。

### 2. 启动服务器

有两种方式启动：

**方式一：同时启动前后端（推荐）**
```bash
npm run dev
```

**方式二：分别启动**
- 终端1：启动后端
```bash
npm run server
```

- 终端2：启动前端
```bash
npm run client
```

### 3. 访问游戏

- 前端地址：http://localhost:3000
- 后端地址：http://localhost:3001

### 4. 开始对弈

1. 打开第一个浏览器窗口，访问 http://localhost:3000
   - 你会自动成为**红方**（先手）
   - 等待对手加入...

2. 打开第二个浏览器窗口（或另一台设备），访问 http://localhost:3000
   - 你会自动成为**黑方**（后手）
   - 游戏开始！

## 使用说明

### 游戏操作
- **选择棋子**：点击己方棋子
- **移动棋子**：先点击要移动的棋子，再点击目标位置
- **重新开始**：点击"重新开始"按钮

### 游戏规则
- 红方先手，黑方后手
- 完整的中国象棋规则已实现
- 包括：车、马、炮、兵、将、士、象的所有走法
- 支持将帅对脸

### 注意事项
- 目前只支持两人对弈（第一个连接的是红方，第二个是黑方）
- 如果对手断开连接，游戏会等待重新连接
- 刷新页面会重新加入游戏

## 部署到服务器

如果你想在不同国家使用，需要将服务器部署到公网：

1. 将代码部署到云服务器（如阿里云、AWS等）
2. 修改 `client/src/App.js` 中的 `SERVER_URL`
3. 确保服务器开放 3001 端口（或你设置的端口）
4. 双方访问部署后的前端地址即可

## 技术栈

- **前端**：React
- **后端**：Node.js + Express + Socket.io
- **通信**：WebSocket 实时通信
